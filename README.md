# TravelBuddy

TravelBuddy is a full-stack MERN travel booking platform developed using MongoDB, Express.js, React.js, and Node.js. The project is designed to provide a centralized travel management system where users can explore destinations, search and book hotels, reserve transportation, manage travel packages, save wishlist items, and discover regional food recommendations.

The application follows RESTful API architecture with a modular backend structure built using Express.js and MongoDB. Mongoose ODM is used for schema modeling, database operations, relationships, validation, and dynamic querying. The backend implements secure JWT-based authentication and bcrypt password hashing to protect user accounts and private routes.

Key features of the platform include hotel listing and booking management, destination management, transportation booking, package reservation, wishlist functionality, food recommendation integration, profile management, and MongoDB-based search and filtering. The system supports dynamic hotel filtering using MongoDB operators such as `$or`, `$gte`, `$lte`, regular expressions, sorting, and ObjectId relationships with population.

Docker and Docker Compose are integrated for containerized deployment, ensuring consistent execution environments and simplified setup across systems. The project demonstrates practical implementation of backend engineering concepts including CRUD operations, middleware handling, REST APIs, MongoDB document modeling, authentication systems, modular architecture, database seeding, and scalable backend workflows.

TravelBuddy is inspired by real-world travel and booking platforms such as Airbnb, Booking.com, Agoda, and MakeMyTrip, and serves as a comprehensive academic and full-stack development project demonstrating modern web application architecture and deployment practices.
